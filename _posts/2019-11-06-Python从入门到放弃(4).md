### 递归
递归中需要有递归公式和中止条件，根据条件中止返回，根据递归公式列函数方程计算阶乘可以使用递归计算

```
def A(n):
    if n==1:
        return 1
    return A(n-1)*n #递归公式n*(n-1)
```
还有杨辉三角等等，最主要的是确定递归公式和终止条件

def fun(a,l):
    for i in range(l):
        for j in range(l-1):
            if a[j]>a[j+1]:
                tmp=a[j]
                a[j]=a[j+1]
                a[j+1]=tmp
    return a

### 整数回文数
给一个整数判断这个整数是否是回文数，不能转化为字符串  
我的思路是转化为一个整数列表然后对比， 还有的思维是直接对比，取到所谓位然后对应位置对比
```
    def isPalindrome(self, x: int) -> bool:
        if x<0:
            return False
        pill=1
        L=[]
        while pill:
            a=10**pill
            if x%a==x:
                L.append(int(x/(a/10)))
                break
            L.append(int((x%a)/(a/10)))
            pill=pill+1
        for i in range(int(len(L)/2)):
            if L[i]!=L[len(L)-1-i]:
                return False
        return True
```
